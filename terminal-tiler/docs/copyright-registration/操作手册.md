## 一、软件概述

### 1.1 软件简介

wise-terminal-tiler 是一款专为 macOS 设计的终端窗口平铺管理工具。它能够帮助用户一键整理散乱的多个终端窗口，按照预设布局自动排列，提升多终端工作流的效率。

### 1.2 核心功能

- **一键整理**: 支持 2~10 个终端窗口的自动平铺
- **多终端支持**: 兼容 iTerm2、Terminal、Ghostty 三大主流终端
- **分区模式**: 可预留屏幕区域给浏览器、聊天工具等其他应用
- **快捷键支持**: 可绑定全局快捷键快速触发
- **AI工具集成**: 支持在 Claude Code、Codex、OpenClaw 中直接调用

### 1.3 适用场景

- 开发人员多终端并行工作
- DevOps 运维多窗口监控
- 需要终端与浏览器并排的工作流
- 多显示器环境下的窗口管理

## 三、安装说明

### 3.1 获取软件

```bash
git clone https://github.com/WiseWong6/wise-labs.git
cd wise-labs
```

### 3.2 执行安装

```bash
mkdir -p ~/.local/bin
./scripts/install-agent-commands
```

安装脚本将自动完成以下操作：
1. 复制主程序到 `~/.local/bin/`
2. 编译 Ghostty 辅助工具
3. 安装 Claude Code 命令文件
4. 安装 Codex 提示词文件
5. 安装 OpenClaw Skills
6. 初始化快捷键绑定

### 3.3 验证安装

```bash
# 检查命令是否可用
tile --help

# 检查快捷键状态
terminal-tile-hotkey status
```

### 3.4 授予权限

首次使用时，系统会提示授予辅助功能权限：

1. 打开 **系统设置** → **隐私与安全性** → **辅助功能**
2. 点击 **+** 按钮添加终端应用（iTerm2 / Terminal / Ghostty）
3. 确保开关处于开启状态

## 四、功能说明

### 4.1 窗口布局矩阵

| 窗口数 | 布局方式 | 说明 |
|--------|----------|------|
| 2 | 2×1 | 双列并排 |
| 3 | 3×1 | 三列并排 |
| 4 | 2×2 | 两行两列 |
| 5~6 | 3×2 | 三列两行 |
| 7~8 | 4×2 | 四列两行 |
| 9 | 3×3 | 三行三列 |
| 10 | 4×3 | 四列三行 |

### 4.2 分区模式说明

分区模式允许用户只整理屏幕的一部分区域，预留另一部分给其他应用：

| 命令 | 区域 | 占比 |
|------|------|------|
| zl2 | 左侧 | 1/2 |
| zl3 | 左侧 | 1/3 |
| zl4 | 左侧 | 1/4 |
| zr2 | 右侧 | 1/2 |
| zr3 | 右侧 | 1/3 |
| zr4 | 右侧 | 1/4 |

### 4.3 智能检测

软件会自动检测当前使用的终端应用：
- 如果在 iTerm2 中执行，只整理 iTerm2 窗口
- 如果在 Terminal 中执行，只整理 Terminal 窗口
- 如果在 Ghostty 中执行，只整理 Ghostty 窗口

可通过环境变量 `TILE_SCOPE=all` 开启跨终端模式。

## 五、使用指南

### 5.1 基本使用

#### 5.1.1 全屏整理

```bash
tile
```

一键整理当前终端应用的所有窗口，平铺到整个屏幕可用区域。

#### 5.1.2 分区整理

```bash
# 左侧半屏
tile zl2
# 或直接使用
zl2

# 右侧三分之一
tile zr3
# 或直接使用
zr3
```

### 5.2 命令列表

| 命令 | 功能 |
|------|------|
| `tile` | 全屏整理 |
| `zl2` | 左侧 1/2 分屏 |
| `zl3` | 左侧 1/3 分屏 |
| `zl4` | 左侧 1/4 分屏 |
| `zr2` | 右侧 1/2 分屏 |
| `zr3` | 右侧 1/3 分屏 |
| `zr4` | 右侧 1/4 分屏 |

### 5.3 环境变量配置

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `TILE_GAP` | 10 | 窗口间距（像素） |
| `TILE_MARGIN_TOP` | 6 | 上边缘留白 |
| `TILE_MARGIN_RIGHT` | 8 | 右边缘留白 |
| `TILE_MARGIN_BOTTOM` | 8 | 下边缘留白 |
| `TILE_MARGIN_LEFT` | 8 | 左边缘留白 |
| `TILE_SCOPE` | current | 整理范围（current/all） |
| `TILE_MODE` | - | iterm_fast 快速模式 |
| `TILE_DEBUG` | 0 | 调试模式（1=开启） |

使用示例：
```bash
# 临时调整间距
TILE_GAP=20 tile

# 开启跨终端模式
TILE_SCOPE=all tile

# 调试模式
TILE_DEBUG=1 tile
```

### 5.4 持久化配置

创建配置文件：

```bash
mkdir -p ~/.config/terminal-window-tiler
cat > ~/.config/terminal-window-tiler/config <<'EOF'
TILE_SCOPE=current
EOF
```

## 六、快捷键配置

### 6.1 默认快捷键

| 功能 | 默认快捷键 |
|------|-----------|
| 全屏整理 | `Ctrl + Command + T` |
| 分区选择器 | `Ctrl + Command + Shift + T` |

### 6.2 查看快捷键状态

```bash
# 查看主快捷键状态
terminal-tile-hotkey status

# 查看分区快捷键状态
terminal-tile-hotkey zone-status
```

### 6.3 修改快捷键

```bash
# 修改主快捷键
terminal-tile-hotkey set cmd+shift+t

# 修改分区快捷键
terminal-tile-hotkey zone-set ctrl+command+shift+t
```

### 6.4 卸载快捷键

```bash
# 卸载主快捷键
terminal-tile-hotkey uninstall

# 卸载分区快捷键
terminal-tile-hotkey zone-uninstall
```

## 七、分区模式详解

### 7.1 二分屏协作模式

适合终端与浏览器/微信并排工作：

```
┌─────────────────┬─────────────────┐
│                 │                 │
│   终端区域      │   浏览器/微信   │
│   (zl2/zr2)     │                 │
│                 │                 │
└─────────────────┴─────────────────┘
```

使用命令：
```bash
zl2  # 终端占左半屏
zr2  # 终端占右半屏
```

### 7.2 三分区协作模式

适合终端、浏览器、聊天工具三开：

```
┌──────────┬──────────┬──────────┐
│          │          │          │
│  终端    │  浏览器  │   微信   │
│ (zl3)    │          │          │
│          │          │          │
└──────────┴──────────┴──────────┘
```

使用命令：
```bash
zl3  # 终端占左 1/3
zr3  # 终端占右 1/3
```

### 7.3 四分屏模式

适合高密度终端工作：

```
┌─────────┬──────────────────────┐
│         │                      │
│ 终端    │     浏览器/其他      │
│ (zl4)   │                      │
│         │                      │
└─────────┴──────────────────────┘
```

使用命令：
```bash
zl4  # 终端占左 1/4
zr4  # 终端占右 1/4
```

## 九、常见问题

### 9.1 窗口没有被整理？

**可能原因及解决**:
1. 未授予辅助功能权限 → 在系统设置中添加终端应用
2. 只有一个终端窗口 → 需要至少 2 个窗口才会触发整理
3. Ghostty 窗口检测失败 → 运行 `ghostty-window-helper list` 检查

### 9.2 快捷键不生效？

**检查步骤**:
1. 确认当前前台应用是终端
2. 运行 `terminal-tile-hotkey status` 检查绑定状态
3. 检查是否有快捷键冲突
4. 尝试重新设置快捷键

### 9.3 Ghostty 窗口无法移动？

**解决方案**:
1. 确保 Ghostty 已被添加到辅助功能权限
2. 检查 `ghostty-window-helper` 是否已编译
3. 运行 `ghostty-window-helper list` 测试连接

### 9.4 多显示器支持？

软件支持多显示器环境，窗口会按照当前所在显示器进行分组整理。每个显示器独立计算布局。

### 9.5 如何卸载？

```bash
# 卸载快捷键
terminal-tile-hotkey uninstall
terminal-tile-hotkey zone-uninstall

# 删除安装的命令
rm -f ~/.local/bin/{tile,zl2,zl3,zl4,zr2,zr3,zr4,terminal-tile-all,terminal-tile-hotkey,terminal-tile-zone-picker,ghostty-window-helper}

# 删除配置文件
rm -rf ~/.config/terminal-window-tiler
```

## 十、技术规格

### 10.1 技术架构

| 组件 | 技术 |
|------|------|
| 主程序 | zsh + AppleScript |
| Ghostty 辅助 | Swift + Accessibility API |
| 快捷键服务 | macOS Automator Services |
| 配置存储 | Plist + 文本文件 |

### 10.2 支持的终端应用

| 终端应用 | Bundle ID | 支持状态 |
|----------|-----------|----------|
| iTerm2 | com.googlecode.iterm2 | ✅ 完整支持 |
| Terminal | com.apple.Terminal | ✅ 完整支持 |
| Ghostty | com.mitchellh.ghostty | ✅ 完整支持 |

### 10.3 布局算法

软件采用智能列布局算法：
- 每列最多 4 个窗口
- 窗口按屏幕分组
- 短列复用满列行高
- 支持从左到右或从右到左展开
